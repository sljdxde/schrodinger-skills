# Forecasting Framework

房价预测只输出区间、情景和置信度，不输出确定性结论。

## 人口驱动模型

学区房需求的长期变量是适龄入学人口：

```text
出生人口 T 年 -> 小学入学 T+6 年 -> 学区房需求 T+5 到 T+6 年
出生人口 T 年 -> 初中入学 T+12 年 -> 初中学区需求 T+10 到 T+12 年
```

简化判断：

```text
未来入学需求变化率 ~= (目标入学年份对应出生人口 - 高峰期出生人口) / 高峰期出生人口
```

应用步骤：

1. 找城市或区县近 10 年出生人口、出生率或小学新生人数。
2. 标出 2016-2017 二孩高峰、2021 后低出生段、当前入学年份。
3. 判断目标学校未来 3-6 年处于高峰、尾声还是回落。
4. 将入学需求变化映射到学区房需求，但不要机械换算成同等房价跌幅。

## 供需和流动性模型

| 指标 | 计算方式 | 解读 |
|---|---|---|
| 供需比 | 在售房源数 / 月均成交量 | 大于 6 个月通常偏买方市场 |
| 议价空间 | 1 - 成交价 / 挂牌价 | 越大说明卖方越弱 |
| 去化周期 | 挂牌量 / 近 12 月均成交量 | 判断流动性 |
| 入学门槛 | 最低可入学总价 | 判断教育溢价和接盘门槛 |
| 换手率 | 年成交套数 / 总户数 | 判断流动性和学位轮转 |

## 价格动量

短期价格受市场情绪和成交量影响：

```text
短期预期价格 = 当前可比成交价 * (1 + 动量系数 * 近期涨跌幅)
```

动量系数建议：

- 上涨趋势：0.3-0.5，趋势延续但衰减。
- 下跌趋势：0.5-0.7，下跌动量通常更强。
- 政策冲击：0.8-1.0，先观察 3-6 个月成交验证。

## 政策冲击矩阵

| 政策 | 短期影响 | 中期影响 | 长期影响 |
|---|---|---|---|
| 多校划片扩大 | 预期混乱、价格波动 | 学区确定性下降 | 教育溢价弱化 |
| 教师轮岗深化 | 心理冲击 | 名校师资独占性下降 | 校际差距收敛 |
| 公民同招 | 择校策略改变 | 公办保底价值变化 | 学区价值重新排序 |
| 六年一学位 | 周转受限 | 投资属性降低 | 流动性折价 |
| 限购/限贷放松 | 需求释放 | 取决于基本面 | 对高价学区支撑有限 |

## 三情景输出

每份报告必须输出三情景：

- 基准：政策和经济没有重大变化，当前人口和成交趋势延续。
- 乐观：利率/限购/产业或学校口碑出现利好，成交量先行改善。
- 悲观：政策确定性下降、出生人口继续下行、库存上升或学校口碑转弱。

## 置信度规则

- 高：近期成交样本充分，政策稳定，多源数据一致。
- 中：成交样本一般，关键数据可核验但有口径差异。
- 低：样本少、政策不确定、学校数据主要来自民间口径。

预测错误最常见原因也要写出来，例如政策突变、利率大幅变化、学校划片调整、区域规划落地超预期、宏观市场快速反转。
