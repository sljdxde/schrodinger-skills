# Report Template

报告要先给结论，再给证据。不要把用户逼着读完 4000 字才知道能不能买。

## 开头结论

```text
# {城市}{小区/片区}购房分析

结论：{建议买入/谨慎可买/继续观望/不建议买入}
适用前提：{自住/学区/改善/投资}
建议价格带：{总价或单价区间}
最大风险：{一句话}
下一步：{实地看房/等成交验证/换备选}
```

## 正文结构

1. 需求和假设：城市、楼盘、预算、入学年份、购房目的、数据截止日期。
2. 楼盘基本面：位置、产品、户型、物业、配套、优缺点。
3. 交易与价格：成交、挂牌、库存、议价、同户型样本。
4. 学校与生源：对口、升学、政策、学位、学校生源和小区人口画像。
5. 横向比较：2-3 个可比楼盘/片区，解释可比原因。
6. 价格预期：三情景、三时间段、置信度和关键假设。
7. 风险清单：政策、市场、流动性、学校、小区品质、家庭现金流。
8. 操作建议：买入条件、谈判抓手、放弃条件、替代方案。
9. 数据来源和局限：列出来源、时间、口径差异。

## 评分表

| 维度 | 权重 | 评分 | 说明 |
|---|---:|---:|---|
| 居住品质 | 20% | | 产品、物业、配套、通勤 |
| 学校确定性 | 20% | | 划片、政策、学位规则 |
| 学校质量与生源 | 20% | | 升学、口碑、生源结构 |
| 市场流动性 | 15% | | 成交、挂牌、换手 |
| 价格安全边际 | 15% | | 跌幅、议价、可比价 |
| 长期人口与供需 | 10% | | 出生、流入、库存 |

评分解释：

- 8-10：价值稳固，适合明确自住需求。
- 6-8：有支撑但需价格安全边际。
- 4-6：风险较高，除非价格明显补偿。
- 4 以下：不建议以学区或投资逻辑买入。

## 购买建议写法

### 建议买入

必须同时满足：

- 价格不高于近期同户型成交中位数，或有明确装修/楼层优势。
- 学校确定性可核验，目标入学年份政策风险可控。
- 家庭现金流能承受 12-24 个月价格不涨或小幅下跌。

### 谨慎可买

适用于自住价值强但市场仍弱的情况。给出谈判价、可接受溢价和止损条件。

### 继续观望

给出观察指标，例如连续 2-3 个月成交量恢复、议价空间收窄、学区政策落地、目标户型出现低于某价位的成交。

### 不建议买入

说明根因：学校不确定、流动性弱、价格仍高估、居住品质无法托底或家庭现金流压力过大。必须给替代方向。
